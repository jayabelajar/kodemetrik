# Code Metric Analyzer (Kodemetrik)

MVP website untuk menganalisis kualitas source code dengan:
- Cyclomatic Complexity (McCabe)
- Halstead Metrics

Support awal: JavaScript dan PHP.

## Development

```bash
npm run dev
```

Open `http://localhost:3000`.

## Notes (MVP)
- Analisis dilakukan client-side (cocok untuk Vercel).
- Limit rekomendasi: max 5MB, max 100 file.
